# 🎬 CineVault

A movie discovery app built with React and the TMDB API.

## Live Demo
## Live Demo
[CineVault](https://tmdb-movies-search-ezj6.vercel.app)

## Features
- Browse popular movies
- Search for any movie with live suggestions
- Filter movies by genre
- View full movie details and watch trailers
- Trending movies this week
- Pagination to browse more movies
- Loading skeletons while data is fetching

## Technologies Used
- React
- Vite
- Axios
- TMDB API
- CSS

## How to run locally

1. Clone the repo
```bash
git clone https://github.com/geraldmrls/TMDB-movies-search.git
```

2. Install dependencies
```bash
npm install
```

3. Create a `.env` file in the root and add your TMDB API key
```
VITE_TMDB_API_KEY=your_api_key_here
```

4. Run the app
```bash
npm run dev
```

## API
This app uses the [TMDB API](https://www.themoviedb.org/documentation/api). You will need a free API key to run it locally.
